# 🖼️ AWS Serverless Image Resizer

A fully serverless image processing application built using **AWS Lambda**, **Amazon S3**, **Lambda Layers (Pillow)**, and **Static Website Hosting**. Users can upload an image, automatically resize it to **300 × 300 pixels**, compare the original and resized images, and download the optimized version.

---

# 🚀 Features

- Drag & Drop Image Upload
- Browse Image Upload
- Automatic Image Resize (300×300)
- Original & Resized Image Preview
- File Size Comparison
- Compression Percentage
- Space Saved Statistics
- Processing Time Display
- Download Resized Image
- Upload Progress Bar
- Upload Speed Indicator
- Responsive Modern UI
- Fully Serverless Architecture

---

# ☁️ AWS Services Used

| AWS Service | Purpose |
|-------------|----------|
| Amazon S3 | Store original & resized images |
| AWS Lambda | Resize uploaded images |
| Lambda Layer | Pillow Image Library |
| S3 Static Website Hosting | Host Frontend |
| IAM | Permissions |

# 🏗 Architecture

```text
                 User
                   │
                   ▼
          S3 Static Website
                   │
            Upload Image
                   │
                   ▼
      uploads/ Folder (Amazon S3)
                   │
          S3 Event Trigger
                   │
                   ▼
             AWS Lambda
          (Python + Pillow)
                   │
           Resize Image
                   │
                   ▼
      resized/ Folder (Amazon S3)
                   │
                   ▼
         Browser Preview
                   │
                   ▼
      Download Resized Image
```

# ⚙ Workflow

1. Upload Image
2. Store image in uploads/
3. S3 triggers Lambda
4. Lambda resizes image
5. Save resized image in resized/
6. Display both images
7. Download optimized image
